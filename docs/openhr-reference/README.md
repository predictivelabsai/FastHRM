# openhr — consolidated reference

This folder consolidates the earlier **openhr** project (a pure-Python HR
platform explored for Nordic / EU public-sector deployments) into **FastHRM**,
which supersedes it as the FastHTML HR demonstrator in the
`fasthtml-oss-migrations` initiative.

Kept here for reference:

- `architecture.md` — openhr reference architecture.
- `oss_crm_comparison.md` — open-source HR/CRM landscape comparison.
- `diagrams/` — architecture / data-flow / integration / AI-stack diagrams (mermaid + PNG).
- `cbre/` — CBRE architecture-choices deck.
- `src-openhr/` — openhr's earlier Python implementation (employee/leave/attendance/payroll), kept as a reference; FastHRM is the current implementation.
- `openhr-README.md` — the original openhr README.

The live app is **FastHRM** (this repo). The original `predictivelabsai/openhr`
GitHub repo remains as history.
